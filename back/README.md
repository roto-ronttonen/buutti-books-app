# About project

Project requries following enviroment variables to be set:

- DATABASE_URL=mongodb://localhost:27017/buutti-books (or wherever your mongodb installation is located)

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
