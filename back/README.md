# About project

Project requries following enviroment variables to be set:

- DATABASE_URL=mongodb://localhost:27017/buutti-books (or wherever your mongodb installation is located)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
